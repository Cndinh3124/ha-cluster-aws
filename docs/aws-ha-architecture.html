🔄 Luồng Request (Request Flow)
1. DNS
User truy cập doanhoitdc.id.vn → PA Việt Nam DNS → CNAME → ALB DNS
2. ALB
ALB nhận request port 80 → Health check → Round Robin → ECS Task
3. ECS
Container PHP+Apache xử lý request → query MySQL nếu cần
4. RDS
MySQL Primary xử lý query → trả kết quả cho PHP
5. Response
PHP render HTML → ALB → User (latency ~50-100ms)
⚡ Failover Timeline (Container Crash)
T+0s
Container 1 (AZ-1a) crash đột ngột
T+10s
ALB health check thất bại → loại Container 1 khỏi pool
T+10s
100% traffic → Container 2 (AZ-1b) · User KHÔNG thấy lỗi
T+30s
ECS phát hiện running < desired → tạo Container mới
T+60s
Container mới RUNNING → ALB thêm vào pool → 2/2 ✅
